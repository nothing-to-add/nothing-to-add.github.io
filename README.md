# Personal Portfolio Website

A modern, responsive personal portfolio website for showcasing iOS development skills and projects.

## Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and form interactions
- **SEO Optimized**: Proper meta tags and structure for search engines
- **Fast Loading**: Optimized images and efficient code
- **Accessible**: Built with accessibility best practices

## Sections

1. **Hero Section**: Introduction with profile photo and key information
2. **About Section**: Detailed information about skills and experience
3. **Projects Section**: Showcase of iOS applications and development work
4. **Contact Section**: Contact form and social media links

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid, Flexbox, and animations)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Inter)
- GitHub Pages for hosting

## Customization

To personalize this website:

1. Replace placeholder content in `index.html`:
   - Update "Your Name" with your actual name
   - Update email and contact information
   - Add your social media links
   - Update project descriptions and links

2. Update the profile photo:
   - Replace `Resources/profile_photo.jpg` with your photo
   - Ensure the image is square (300x300px or larger)

3. Customize colors in `styles.css`:
   - Main brand color: `#2563eb`
   - Gradient colors in hero section
   - Adjust spacing and typography as needed

4. Update `_config.yml` with your information:
   - Site title and description
   - Your email and social media handles
   - GitHub username

## Local Development

To run this site locally:

1. Clone the repository
2. Open `index.html` in your browser
3. Or use a local server like Live Server in VS Code

## Deployment

This site is configured for GitHub Pages:

1. Push your changes to the `main` branch
2. Enable GitHub Pages in repository settings
3. Your site will be available at `https://username.github.io`

## iOS Developer Profile

This template is specifically designed for iOS developers and includes:

- Dedicated sections for App Store apps
- Skills showcase with iOS-specific technologies
- Project status indicators (Published, Submitted, In Progress)
